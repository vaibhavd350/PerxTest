# README
## System Dependencies
- Ruby 2.7.5
- Postresql 13.0

## Development Setup
1. Clone this repository
	 - git clone https://github.com/vaibhavd350/PerxTest
2. `bundle install`
3. `rails db:create` 
4. `rails db:migrate`
5. `rails db:seed`
6. `rails server`

## Running Tests
- `rspec` to run the test all the scripts to reduce the issues
- `rspec file_path` to test the specific test scripts
